## Design System

The portfolio uses a **CSS custom properties** approach with two themes:

- **Light**: `#f5f5f5` background, `#333` text, `#1976d2` primary
- **Dark**: `#0a0a0a` background, `#e0e0e0` text, `#64b5f6` primary

Typography uses fluid sizing with `clamp()` for responsive scaling without breakpoints.

## Architecture

```
portfolio-gatsby/
├── src/
│   ├── components/       ← React components (this Storybook)
│   │   ├── home/         ← Landing page sections
│   │   └── __mocks__/    ← Test data for stories
│   ├── context/          ← ThemeContext (light/dark)
│   └── pages/            ← Gatsby pages
├── .storybook/           ← Storybook config + Gatsby mocks
└── docs-site/            ← Docusaurus architecture docs
```
